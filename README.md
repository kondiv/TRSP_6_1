# Практическое задание №1

## Запуск
Для запуска необходимо склонировать репозиторий, перейти в папку проекта Web Api (```>cd Api```) после этого для запуска выполнить команду   
| Протокол| Команда                          |
|---------|----------------------------------|
| http    | dotnet run                       |
| https   | dotnet run --launch-profile https|

## После запуска
После запуска приложение будет доступно по следующим адресам [http://localhost:5296](http://localhost:5296) | [https://localhost/7252](https://localhost/7252)

## Эндпоинты

| Метод   | Адрес                                |
|---------|--------------------------------------|
| POST    | http://localhost:5296/api/items      |
| GET     | http://localhost:5296/api/items      |
| GET     | http://localhost:5296/api/items/{id} |
